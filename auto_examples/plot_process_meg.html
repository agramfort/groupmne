Multi-subject MEG processing¶

The aim of this tutorial is to show how to process MEG with the purpose of computing source estimates jointly for a group of subjects (see Group Lasso example).

import os
import os.path as op
from mne import compute_covariance, write_cov, find_events, Epochs, pick_types
from mne.io import read_raw_fif
from mne.datasets import hf_sef
from matplotlib import pyplot as plt
